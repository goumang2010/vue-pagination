vue-pagination
-------------------

Pagination component for [Vue.js](vuejs.org).

** Compatibility Note **: vue-pagination requires Vue.js 1.0.0+



Installation

* Script Tag
    
    ```js
    <script type="text/javascript" src="http://cdn.bootcss.com/vue/1.0.26/vue.js"></script>
    <script type="text/javascript" src="http://cdn.bootcss.com/vue/1.0.26/vue.js"></script>
    ```
